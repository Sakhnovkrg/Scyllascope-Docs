# Hotkeys

A quick reference of all keyboard and mouse shortcuts in Scyllascope.

## Waveform Area

| Shortcut | Action | Details |
|----------|--------|---------|
| [[Mouse Wheel]] | Zoom in/out at cursor | [Zoom & Navigation](/main/zoom) |
| [[Shift]] + [[Mouse Wheel]] | Scroll horizontally | [Zoom & Navigation](/main/zoom) |
| [[Left Click]] + Drag | Zoom and pan | [Zoom & Navigation](/main/zoom) |
| [[Ctrl]] / [[⌘ Cmd]] + Drag | Select time region | [Time Selection](/main/time-selection) |
| [[Ctrl]] / [[⌘ Cmd]] + [[Click]] | Pin dB label | [dB Ruler](/main/db-label) |
| [[Right Click]] | Open context menu | [Context Menu](/main/context-menu) |
