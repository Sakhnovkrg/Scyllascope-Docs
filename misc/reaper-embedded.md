# Reaper Embedded Mode

![Reaper Embedded Mode](/images/14.gif)

Scyllascope supports Reaper's embedded FX interface, allowing you to display the oscilloscope directly in the **TCP** (Track Control Panel) or **MCP** (Mixer Control Panel) without opening a separate plugin window.

::: warning
Embedded mode is only available in the **VST3** version of the plugin.
:::

## How to enable

1. Add Scyllascope to a track's FX chain.
2. Right-click the **FX insert** on the track.
3. Select **Show embedded UI in TCP** or **Show embedded UI in MCP**.

The oscilloscope will appear directly in the track panel or mixer strip. You can resize the embedded area by dragging the track height (TCP) or strip width (MCP).
